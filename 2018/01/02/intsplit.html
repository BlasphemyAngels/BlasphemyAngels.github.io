分苹果问题

m个苹果放入n个盘子里面，问有多少种分法，不同顺序如1,1,5和5,1,1算一种分法。

解析

设f(m,n)为m个苹果放入n个盘子的方案数。

  1. 当n>m时，则至少有n-m个盘子空着，所以拿掉这n-m个盘子对结果无影响，即f(m,n)=f(m,n-m)；
  2. 当n<=m时，有两种情况：
    1. 至少有一个盘子空着，则方案数为f(m,n-1);
    2. 没有盘子空着，也就是每个盘子都有至少一个苹果，那么将每个盘子都拿走一个苹果，不影响方案计数，所以此时方案数为f(m-n,n); 最终得到动态转移方程：
if(n > m)
    f(m, n) = f(m, n - m);
else
    f(m, n) = f(m, n - 1) + f(m - n, n);

int f[11][11];
for (int i = 1; i <= 10; ++i)
{
    f[i][1] = 1;
    f[1][i] = 1;
    f[i][0] = 1;
    f[0][i] = 1;
}
f[0][0] = 1;
for (int i = 2; i <= 10; ++i)
{
    for (int j = 2; j <= 10; ++j)
    {
        if (j > i)
        {
            f[i][j] = f[i][i];
        }
        else {
            f[i][j] = f[i - j][j] + f[i][j - 1];
        }
    }
}
